## Element Annotations

| Element | Description | Interaction |
|---------|-------------|-------------|
| Statistics Summary | Cards showing total time, count, average | Tap for detailed analytics |
| Time Filter Tabs | Today/Week/All quick filters | Tap to filter sessions |
| Filter Icon | Opens filter/sort sheet | Tap to open modal |
| Section Header | Groups sessions by date | Read-only |
| Session Item | Individual session card | Tap for detail view |
| Session Duration | Duration badge on right side | Read-only |
| Session Time | Start and end time range | Read-only |
| Load More | Pagination trigger | Tap or scroll to load |
| Edit Button | In detail view | Tap to edit session title |
| Delete Button | In detail view | Tap to delete with confirmation |

---

## Design Notes

1. **Grouped Lists**: Use iOS-style grouped list with section headers by date
2. **Statistics Cards**: Use subtle shadows and rounded corners
3. **Swipe Actions**: Support swipe-to-delete on session items
4. **Pull to Refresh**: Standard iOS pull-to-refresh pattern
5. **Skeleton Loading**: Show skeleton states while loading data
6. **Accessibility**: Ensure all stats have proper labels for VoiceOver
