## Element Annotations

| Element | Description | Interaction |
|---------|-------------|-------------|
| Default Duration | Sets timer default when starting session | Tap opens duration picker |
| Session Name | Default name for new sessions | Tap opens text input |
| Sound Toggle | Enable/disable completion sound | Toggle switch |
| Vibration Toggle | Enable/disable haptic on complete | Toggle switch |
| Sound Selection | Choose completion sound | Opens sound picker |
| Live Activity Toggle | Master toggle for Live Activities | Toggle switch |
| Export History | Export sessions as CSV/JSON | Opens share sheet |
| Clear All History | Delete all session data | Shows confirmation alert |
| Version | App version number | Read-only |
| Send Feedback | Opens email/feedback form | Opens mail composer |

---

## Design Notes

1. **Grouped Sections**: Use iOS-style grouped table view
2. **Toggle Switches**: Standard iOS switch components
3. **Destructive Actions**: Red text for delete actions
4. **Haptic Feedback**: Light haptic on toggle changes
5. **Persistence**: All settings persist to AsyncStorage
6. **Dark Mode**: Full support with appropriate contrast

---

## Implementation Priority

**MVP (Phase 1):**
- Default duration setting
- Live Activity toggle

**Post-MVP:**
- Sound selection
- Export history
- Clear history
- Feedback mechanism
