# Timer Screen Wireframe

## Overview
The main timer screen is the primary interface for the Study Timer app. It features a large circular timer display, session controls, and quick access to common durations.

## Element Annotations

| Element | Description | Interaction |
|---------|-------------|-------------|
| Session Title | Displays current session name | Tap to edit via text field |
| Timer Display | Large MM:SS format, centered | Read-only during session |
| Progress Ring | Circular indicator around timer | Animates clockwise as time elapses |
| Duration Presets | 25/45/60 minute quick buttons | Tap to set timer duration |
| Start Button | Primary CTA, green color | Initiates timer countdown |
| Pause Button | Yellow/orange color | Pauses active timer |
| Stop Button | Red color, destructive | Ends session, saves to history |
| Day Stats | Shows today's total time & count | Tap to navigate to history |
| Tab Bar | Navigation between main screens | Standard iOS tab bar |

---

## Design Notes

1. **Timer Ring**: Should use a gradient from primary color to secondary
2. **Haptic Feedback**: Provide haptic feedback on button taps
3. **Animations**: Smooth pulse animation when timer completes
4. **Accessibility**: Large touch targets (44pt minimum), VoiceOver support
5. **Dark Mode**: Full support with inverted color scheme
